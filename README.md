#jQuery Hotkeys

Usage:

 ````html
<script src="/jquery/jquery.js"></script>
<script src="/jquery-hotkeys/jquery-hotkeys.js"></script>
 ````

 ````javascript
$(document).hotkey('ctrl+c', function() {});
$(document).hotkey('ctrl+alt', function() {});
$(document).hotkey('enter+f1', function() {});
````
